Async image loader

Asynchronous and lazy image loading library.

Download this project as a .zip file Download this project as a tar.gz file

How to

After initializing Async image loader will only target elements with data-src attribute.

![](images/loader.svg)
img class="how-to" src="images/loader.svg" data-src="images/check.png" />
script>
  new AsyncImageLoader('.how-to', {bind: [{target: '.how-to', type: 'mouseover'}]});
/script>

Callbacks

Async image loader implements number of callbacks allowing better image manipulation.

img class="callback border" data-src="images/check.png" />
script>
  new AsyncImageLoader('img.callback', {
    bind: [{target: 'img.callback', type: 'mouseover'}],
    callbacks: {
      load: function () {
        $(this).removeClass('border');
      }
    }
  });
/script>

Lazy load

Async image loader can be configured to load images only after they enter viewport.

![](images/loader.svg)
img class="lazy-load" src="images/loader.svg" data-src="images/check.png" />
script>
  new AsyncImageLoader('img.lazy-load', {
    bind: [ { target: window, type: 'scroll', delay: 500 } ],
    conditions: { within_bounds: true }
  });
/script>
