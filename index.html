Async image loader

Asynchronous and lazy image loading library.

View on GitHub

How to

After initializing Async image loader will target only image elements with data-src attribute.

<img class="how-to" src="images/loader.svg" data-src="images/check.png" />
<script>
  new AsyncImageLoader('.how-to', {bind: [{target: '.how-to', type: 'mouseover'}]});
</script>

Lazy load

Async image loader can be configured to load images only after they enter viewport.

<img class="lazy-load" src="images/loader.svg" data-src="images/check.png" />
<script>
  new AsyncImageLoader('img.lazy-load', {
    bind: [ { target: window, type: 'scroll', delay: 500 } ],
    conditions: { within_bounds: true }
  });
</script>
