Async image loader

Asynchronous and lazy image loading library.

Download as .zip Download as .tar.gz View on GitHub

How to

After initializing Async image loader will target only image elements with data-src attribute.

<img src="images/placeholder.gif" data-src="checkpng" />
<script>
  new AsyncImageLoader('img', {bind: [{target: window, type: 'scroll'}]});
</script>
